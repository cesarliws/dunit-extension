Compatibility instructions

en
-------------------------------------------------------------------------------------------------------------------
Dunit-extension requires that the DUnit framework be downloaded from svn trunk
(http://dunit.svn.sourceforge.net/svnroot/dunit), because the latest file avaliable 
for download(dunit-9.3.0.zip) does not contain a unit(CIXMLTestRunner) that generates the Xml Report for Continous 
integration tools.
The "dunit\private\plandolt" directory should be included in the Delphi library search path.

pt-br
-------------------------------------------------------------------------------------------------------------------
DUnit-extension requer que o framework DUnit seja baixado do svn trunk
(http://dunit.svn.sourceforge.net/svnroot/dunit), porque o último arquivo disponível
para download(dunit-9.3.0.zip) não contém a unit(CIXMLTestRunner) que gera o relatório Xml para ferramentas de 
integração contínua.
O diretório "dunit\private\plandolt" deve ser incluído no library path do Delphi.
